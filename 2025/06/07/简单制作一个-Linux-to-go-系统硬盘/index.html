简单制作一个 Linux to go 系统硬盘

发表于2025-06-07|更新于2025-06-07|Linux
|浏览量:|评论数:

0x00 前言

Linux 是一个开源的操作系统，它的设计目标是提供一个稳定、安全、高效的操作系统，适用于各种硬件平台。Linux 可以运行在各种硬件平台上，包括个人电脑、服务器、嵌入式设备等。

你是否有在不同的设备上编写代码的需要，但是不想重新配环境，又不想用对资源占用较高的 Windows 系统？那么你需要一个 Linux to go！

0x01 物品准备

  • 一个U盘（用于制作启动盘，建议16G）
  • 一个 USB 3.0 移动硬盘（用于安装系统，建议固态硬盘或固态 U 盘，128G及以上）
  • 一台能用的电脑

0x02 制作启动盘

1.选择系统

我使用的是 Zorin OS 17.3。

这款系统基于 Ubuntu 22.04，拥有 Ubuntu 的生态，而且运行速度很快的同时拥有漂亮的类 Windows 桌面，是一个很适合 Linux 新手的系统。

2.下载系统镜像

进入 Zorin OS 官网下载页面，找到 Zorin OS 17.3 Core，点击下方 download 按钮。

之后耐心等待下载完成。

3.刻录镜像文件到启动盘

可以根据这篇教程。

0x03 安装系统

请原谅我的拍屏。

插入启动盘，重启，狂按功能键使用刚才的 U 盘启动。

等待启动完毕，在弹出窗口左侧找到并选择中文，然后点击安装 Zorin OS。

然后键盘布局左边选中文，右边选第二个。

然后连接无线网（可选）。

然后按照图片选。

之后在安装类型界面选择擦除整个磁盘安装。

再选择准备好的移动硬盘，继续安装。

之后设置账户。

耐心等待安装完成，然后点击现在重启。

出现图中小字后，拔掉启动 U 盘，按回车。

重启后，就可以进入 Zorin OS 桌面了！

end。
